# knowledge/ — your reference + originality anchor

Drop your channel's reference material here. It is read by the `ideator` and `writer`
sub-agents; it never leaves your machine.

- **Reference scripts** — a handful of competitor/self narration scripts (plain `.txt`).
  The writer matches their voice; the ideator reverse-engineers their premises. These
  are your tone anchor.
- **Channel screenshots** (`*.png` / `*.jpg`) — YouTube grid screenshots showing
  **title + view count + upload age** per tile. These are the virality signal + the
  thumbnail-style reference the ideator reads.
- **`used-scenarios.txt`** — the anti-repeat ledger (managed automatically; don't edit
  by hand).

This folder ships empty on purpose — it holds *your* research, not the kit's.
